# Supply Chain Optimisation using LP and MILP

## Project Overview

This project applies optimisation modelling to support supply chain distribution decisions. A Linear Programming (LP) model is developed to minimise transportation cost across factories, warehouses, and retailers while satisfying capacity and demand constraints.

The model is then extended into a Mixed Integer Linear Programming (MILP) framework by incorporating binary facility activation decisions, fixed operating costs, and carbon emission costs. This allows the project to evaluate both cost efficiency and sustainability-aware supply chain planning.

## Business Problem

Supply chain networks often involve complex decisions about where products should be shipped from, which routes should be used, and how facilities should be utilised. The aim of this project is to identify a cost-efficient distribution strategy while considering operational constraints and environmental cost implications.

## Objectives

- Minimise total transportation cost
- Satisfy factory capacity, warehouse capacity, and retailer demand constraints
- Model shipment flows between factories, warehouses, and retailers
- Extend the basic LP model into a MILP model
- Include facility activation decisions using binary variables
- Incorporate fixed operating costs and carbon emission penalties
- Generate managerial insights from optimisation results

## Methodology

The project uses Excel Solver to develop and solve two optimisation models:

1. **Linear Programming Model**
   - Optimises shipment quantities
   - Minimises transportation cost
   - Applies capacity, flow-balance, and demand constraints

2. **Mixed Integer Linear Programming Model**
   - Extends the LP model with binary open/close facility decisions
   - Includes fixed facility costs
   - Incorporates carbon emission cost
   - Evaluates cost and sustainability trade-offs
